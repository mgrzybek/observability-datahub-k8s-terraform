# strimzi-operator

Installs the Strimzi Operator using the Openshift marketplace.

## Usage

### Openshift

```hcl
module "operator" {
  source = "./strimzi-operator"

  isOpenshift     = true
  operatorSource  = "community-operator"
  sourceNamespace = "openshift-marketplace"
  startingCSV     = "strimzi-cluster-operator.v0.31.1"
} 
```

### Kubernetes

```hcl
module "operator" {
  source = "./strimzi-operator"
} 
```
